## (1) 프로젝트 명칭
머니로그 (MoneyLog): 심플한 개인 자산 관리 도구

## (2) 개발 시스템 명칭
MoneyLog system

## (3) 개발 시스템 설명
개인이 매일의 수입과 지출을 빠르게 기록하고 확인하기 위한 모바일 앱입니다.
사용자가 자신의 소비 습관을 한눈에 파악할 수 있도록 월별 통계를 제공합니다.
외부 은행 연동 없이 사용자가 직접 입력하는 수동 입력 방식을 채택하여 개인정보 유출 걱정 없이 안전하게 사용할 수 있습니다.

## (4) 개발 시스템 핵심 기능
지출/수입 내역 입력: 금액, 날짜, 카테고리(식비, 교통비 등)를 간단히 저장하는 기능.
내역 조회 및 필터링: 날짜별 또는 카테고리별로 과거 기록을 찾아보는 기능.
월간 요약 리포트: 한 달 동안 총 얼마를 썼고 남았는지 수치로 보여주는 통계 기능.
카테고리 커스터마이징: 사용자가 직접 원하는 카테고리 이름을 추가하거나 수정하는 기능.
예산 설정 및 알림: 이번 달 쓸 목표 금액을 정해두고, 초과 시 경고를 띄워주는 기능.

## (5) 기존 유사 소프트웨어 분석 및 차별점
유사 서비스: 토스(Toss), 뱅크샐러드.
차별점
오프라인 우선: 은행 연동이 강제되는 기존 앱과 달리, 모든 데이터를 내 기기에만 저장하여 프라이버시를 극도로 보호함.
미니멀리즘: 복잡한 금융 상품 광고나 부가 기능 없이 오직 '기록'과 '통계'에만 집중하여 앱이 매우 가볍고 빠름.
